Fibonacci sequence Open on JetBrains Academy

Fibonacci sequence is a sequence of numbers such that each element is a sum of the previous two, and the first two elements are equal to 0 and 1.

Define a generator that produces the Fibonacci sequence of the given length n.

You don't have to handle the input, just implement the generator function called fibonacci.


Sample Input:
5
Sample Output:
[0, 1, 1, 2, 3]


Memory limit: 256 MB
Time limit: 15 seconds
